Installation
============

From Source
-----------

.. code-block:: bash

    git clone https://github.com/yourusername/pr-collector.git
    cd pr-collector
    make install

Development Installation
------------------------

.. code-block:: bash

    git clone https://github.com/yourusername/pr-collector.git
    cd pr-collector
    make install-dev

Requirements
------------

* Python 3.12+
